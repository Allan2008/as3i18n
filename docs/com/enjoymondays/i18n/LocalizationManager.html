addSupporetedCodes()method
public function addSupporetedCodes(... langs):void

Set the supported locales, usually on initializing ILocalizationManager instance.

Its a convinience method that calls multiple times addSupportedLocale We can pass any value that resolves to a valid ILocale instance.

         var locales:Array = [Locale.ES, Locale.CA, "en" ];
         LocalizationManager.instance.addSupporetedCodes( locales );
         //or
         LocalizationManager.instance.addSupporetedCodes( Locale.ES, Locale.CA, "en" );

Parameters

... langs — An array containing ILocale instances of supported languages.
